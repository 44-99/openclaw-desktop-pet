## 🚀 使用说明

### 左键操作
- **短按**（<200ms）：生成新话题
- **长按拖动**：移动窗口位置

### 右键操作
- **单击**：打开菜单
- **长按拖动**：旋转哈基虾 360°

### 菜单功能
- 🦞 打开 OpenClaw → 使用默认浏览器打开当前话题
- 🔄 360° 查看我 → 旋转哈基虾
- 📊 系统状态 → 显示当前状态
- 👻 隐藏 → 隐藏窗口
- ❌ 退出哈基虾 → 完全退出
